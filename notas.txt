Autenticación: cifrar contraseñas para evitar problemas de seguridad

-La dependencia más usada para poder cifrar contraseñas es bcrypt.
-Esta la usaremos para cifrar y descifrar el pass para no guardarlo en limpio.
-Esta hasheara el pass, y lo comparárá con el creado para ver sí es el correcto.

-Para cifrar un password, usaremos el metodo hash(), con 2 params, el password y 
  la cantidad de veces que se repetirá el algoritmo. Mientras más veces se repita 
  más lento será generandolo, pero será más seguro. El metodo devuelve una promesa
  con el pass cifrado.
-Para comparar sí un hash es equivalente a un password, se usará el metodo compare,
  que devuelve una promesa con un booleano indicando sí son equivaletes.

-En el upsert, al colocar la contraseña, usaremos hash(). Eto
-En el login, usaremos compare(), con el pass que viene del body y el del objeto
  con los datos, y sí funciona, ahí realizaremos el sign() para traer el token.
